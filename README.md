# ARCH.SYS — Portfolio

Portfolio personal + negocio + blog para un profesional de DevOps, Cybersecurity e IA.

## Stack
- React 18 + React Router v6
- Vite 5
- CSS Modules

## Inicio

```bash
npm install
npm run dev
```

## Rutas
| Ruta | Página |
|------|--------|
| `/` | Home |
| `/portfolio` | Galería completa |
| `/servicios` | Servicios |
| `/blog` | Blog |
| `/blog/:slug` | Post |
